# Monty App Icon Source

The app icon is derived from `Bernard_Montgomery_portrait_1943_TR1042.jpg`, downloaded from Wikimedia Commons:

- Source page: <https://commons.wikimedia.org/wiki/File:Portrait_of_the_Eighth_Army_Commander,_General_Sir_Bernard_Montgomery,_wearing_his_%27battle_sweater%27_and_tank_beret,_in_the_UK,_1943._TR1042.jpg>
- Description: Portrait of General Sir Bernard Montgomery, wearing his battle sweater and tank beret, in the UK, 1943
- Date: 29 May 1943
- Source institution: Imperial War Museums
- Photographer: Edward G. Malindine / War Office official photographer
- License status on Wikimedia Commons: Public domain as an expired United Kingdom Government / Crown copyright work, marked `PD-scan` / `PD-UKGov`
- Downloaded file SHA-256: `7983952380b55543c511bf2275a583dafab079b4ba7696e4003fff0f733b2916`

The icon crop focuses on Montgomery's face and beret while avoiding the uniform jacket insignia as the visual subject.
